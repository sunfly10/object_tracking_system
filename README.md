1. 서론

  본 프로젝트는 단거리 지대공 유도무기인 '천마'의 시선 지령 유도 방식에서 착안하여, 비전 기반의 실시간 객체 추적 시스템을 구현한 개인 프로젝트입니다.

  탐지/추적 레이더와 유도탄 발사 장치의 메커니즘을 카메라와 2축 서보모터로 재해석하여, 타겟 객체를 화면 중앙(조준선)에 지속적으로 유지하는 자동 추적 시스템을 구축했습니다.

  •개발 환경: Ubuntu 22.04, ROS 2 (Humble)

  •주요 언어: Python, C++(Arduino)

  •하드웨어: PC, Raspberry Pi 5, Arduino mega 2560, Servo Motor(sg-90) 2개, Raspberry Pi Camera

2. System Architecture

  단일 보드의 연산 리소스 한계를 극복하고 실시간성을 확보하기 위해, 이기종 디바이스 간 역할을 분담한 분산 제어 시스템을 설계했습니다.

  •PC (Vision & AI): 고해상도 영상에 대한 YOLO 객체 인식 추론 수행

  •Raspberry Pi (Master): 카메라 영상 취득 및 Flask 스트리밍 서버 운용, 객체 특징 데이터(중심 좌표, 면적) 산출, 제어 오차 연산 및 ROS2 노드 통신 총괄

  •Arduino (Actuator): 수신된 목표 각도 데이터를 기반으로 서보모터 구동 및 현재 상태 피드백

  3. Key Technologies & Algorithms

  •Vision 기반 조건부 타겟 필터링:

  YOLO가 탐지한 바운딩 박스를 바탕으로 객체의 클래스 이름, 면적(거리 추정), 수평/수직 위치 4가지 파라미터를 도출합니다.

  다수의 객체 중 사용자가 설정한 조건을 모두 만족하는 단일 객체만을 추적 타겟으로 고정하여 시스템의 안정성을 확보했습니다.

  •정밀 모터 제어 (Variable Gain PID & Anti-Windup):

  카메라 화각 정규화: 픽셀 오차를 맹목적으로 추종하지 않고, 카메라의 수평 시야각(54도) 특성을 반영하여 실제 제어에 필요한 각도 단위로 정규화했습니다.

  가변 이득 PID 제어: 오차 크기(2도 기준)에 따라 PID 제어 게인 값을 동적으로 변경했습니다. 오차가 클 때는 Ki, Kd 값을 높여 정지 마찰력을 극복하고 신속히 이동하며, 오차가 작을 때는 게인을 낮춰 오버슈트와 미세 진동을 방지했습니다.

  Anti-Windup 적용: 적분 항이 과도하게 누적되어 발생하는 발산 현상을 막기 위해, 적분 오차의 범위를 ±30으로 제한하는 Clamping 기법을 적용했습니다.

  •시스템 최적화 및 동적제어

  멀티스레드 및 동기화: ROS 2 통신과 Flask 스트리밍을 병렬 처리하며, 공유 이미지 프레임 접근 시 Lock 동기화를 적용해 데이터 무결성을 확보했습니다.

  비동기 처리: 전체 시스템 지연을 방지하기 위해 0.4초 주기의 독립적인 타이머 인터럽트로 YOLO 연산을 분리했습니다.

  런타임 동적 재설정: ROS 2 Parameter Server(add_on_set_parameters_callback)를 활용하여 시스템 재시작 없이 런타임 중에 타겟의 이름, 크기, 위치 조건을 실시간으로 변경할 수 있는 인터페이스를 구축했습니다.

  양방향 프로토콜 통신: PC와 아두이노 간 ASCII 문자열 기반의 Command/Feedback 프로토콜(예: x:90, receive_x)을 구축하여, 개방 루프의 불확실성을 제거한 폐루프 제어를 구현했습니다.

  4. 결과

  목표 객체가 이동할 경우 시스템이 허용 오차 범위 내로 조준선을 재정렬하는 데 걸리는 정착 시간(Settling Time)을 3초 이내로 달성했습니다.

  가변 이득 PID 제어를 통해 초기 오버슈트를 최소화하고 타겟을 안정적으로 중앙에 위치시키는 결과를 그래프를 통해 검증했습니다.

  5. 한계점 및 개선점

  기구학적 한계 극복: 구동 시 발생하는 모터 반작용력으로 인해 정지 상태에서 미세 잔류 진동이 발생했습니다. 향후 3D 프린터로 전용 결합 브라켓을 설계하여 물리적 유격을 제거할 계획입니다.

  필터 기반 노이즈 제거: 비전 모델이 출력하는 바운딩 박스 자체의 미세한 흔들림을 보정하기 위해 이동 평균 필터(Moving Average Filter) 또는 칼만 필터(Kalman Filter)의 도입이 필요함을 확인했습니다.

  지연 보상을 위한 제어 고도화: 영상 처리 및 통신에서 발생하는 필연적인 시간 지연을 보상하기 위해, 객체의 이동 벡터를 산출하여 미래 위치를 추종하는 피드포워드(Feed-forward) 알고리즘의 필요성을 도출했습니다.
